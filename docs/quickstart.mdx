## Not Sure Which Guide to Use?

Here's a quick decision tree:

**Are you writing code?**
- **Yes** → Continue below
- **No** → Check out [Google Tag Manager](/integrations/gtm), [Shopify](/integrations/shopify), or [WordPress](/integrations/wordpress)

**What framework are you using?**
- **React** → [React Quick Start](/quickstart/react)
- **Next.js** → [Next.js Quick Start](/quickstart/nextjs)
- **Vue, Svelte, Angular** → [Vanilla JS Quick Start](/quickstart/vanilla-js)
- **None / Plain HTML** → [CDN Quick Start](/quickstart/cdn)
- **Backend / Server** → [Node.js Quick Start](/quickstart/nodejs)

## What's in Each Guide?

Each platform guide includes:

- **Installation steps** specific to that platform
- **Code examples** you can copy and paste
- **Best practices** for that environment
- **Common patterns** and pitfalls to avoid

All guides take about 5 minutes to complete.
